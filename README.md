# Docker-compose.yml-DZ
## Инструкция по запуску программы
Создайте папку в документах. В терминале перейдите в созданую вами папку, с помощью команды **git clone** создайте локальную копию репозитория. Перейдите в папку репозитория **Docker-compose.yml-DZ**. Пропишите команду **sudo docker compose up  --build**, это запустит мой контейнер с программой. Для останвки и выхода пропишите sudo **docker compose down**.
